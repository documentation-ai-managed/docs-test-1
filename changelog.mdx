## Migration Guide

If you use deprecated features from v1.1.0 and earlier, follow these steps to upgrade seamlessly.

<Steps>
  <Step title="Review Deprecated Endpoints" icon="alert-triangle">
    Check your integration for `/v1/projects` calls.
  </Step>
  <Step title="Update to v2 API" icon="code">
    Replace with `/v2/projects`. See code example above.
  </Step>
  <Step title="Test in Sandbox" icon="play">
    Use `https://sandbox.example.com` for validation.
  </Step>
  <Step title="Deploy and Monitor" icon="rocket">
    Go live and watch for webhook events.
  </Step>
</Steps>

<Callout kind="tip">
Run `npm update testio-sdk` or equivalent to get the latest client libraries.
</Callout>
